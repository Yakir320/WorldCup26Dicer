Per-model shape · the "magic numbers"
ρ tweaks low-score/draw balance · r is Neg-Binom spread (high r ≈ Poisson) · ν is COM-Poisson tightness (ν>1 fits football's clustered scores). Calibrate finds the values that best fit the finished games — but that's in-sample on ~30 games, so it overfits (it'll push ν high). Treat it as a starting point and sanity-check in the Model report.
